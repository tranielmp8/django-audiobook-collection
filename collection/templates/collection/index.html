Welcome to The AudioBook Collection App!

Create, Read, Update, and Delete from your very own audiobook collection.

Listening to Audiobooks is a hobby of mine. If it is yours also then you will love this app. This app allows you to create a list of audiobooks you have listened to. In this list you can upload an image for each audiobook plus write your own description of the book. There are a few other features you can attach to your books.
